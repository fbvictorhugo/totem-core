# Contribuindo com o TotemCore

Obrigado pelo interesse em contribuir com o **TotemCore**!
Este projeto busca criar uma base open source para sistemas de coleta de informações através de totens digitais, com foco em iniciativas comunitárias, eventos e projetos sociais.

Toda contribuição é bem-vinda.

---

## Como Contribuir

Você pode contribuir de várias formas:

* Reportando bugs
* Sugerindo melhorias
* Melhorando a documentação
* Implementando novas funcionalidades
* Corrigindo problemas no código

---

## Reportando Problemas

Se encontrar um problema, abra uma **Issue** descrevendo:

* o problema encontrado
* como reproduzir o erro
* comportamento esperado
* screenshots ou logs (se possível)

Isso ajuda bastante na identificação e correção do problema.

---

## Sugerindo Melhorias

Sugestões são muito bem-vindas.

Antes de implementar uma nova funcionalidade, recomendamos abrir uma **Issue** para discutir a ideia com os mantenedores do projeto.

---

## Contribuindo com Código

Se quiser contribuir com código:

1. Faça um **fork** do repositório
2. Crie uma nova **branch** para sua alteração
3. Faça suas modificações
4. Commit suas alterações
5. Abra um **Pull Request**

Exemplo de fluxo:

```
Fork → Branch → Commit → Pull Request
```

---

## Boas Práticas

Ao contribuir com o projeto, procure:

* manter o código claro e organizado
* seguir o estilo de código já utilizado no projeto
* escrever commits descritivos
* manter mudanças pequenas e focadas
